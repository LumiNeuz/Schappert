# Manuel Schappert Personenbeförderung - Website PRD

## Problem Statement
Erstelle eine moderne, professionelle Website für Manuel Schappert Personenbeförderung in Meisenheim. Die Seite soll klar, ansprechend und nutzerfreundlich sein.

## User Personas
- **Patienten**: Senioren und Menschen für Krankenfahrten, Dialyse, Chemo-/Strahlentherapie
- **Mobilitätseingeschränkte**: Rollstuhlfahrer, barrierefreie Transporte
- **Familien**: Eltern für Schülerfahrten
- **Lokale Kunden**: Meisenheim und Umgebung für Kurierfahrten

## User Choices (Dezember 2025)
- Helles, freundliches Design (weiß/creme)
- Kein Kontaktformular, nur Kontaktdaten
- Klickbare Telefonnummer für mobile Nutzer
- Stock-Bilder für Fahrzeuge/Transport
- Nur Deutsch

## Core Requirements (Static)
- Single Page Landing mit Hero, Leistungen, Über uns, Warum wir, Kontakt
- Impressum und Datenschutz Seiten (deutsche Rechtspflicht)
- Mobile-first responsive Design
- SEO-optimierte Texte
- Barrierefreiheit (hoher Kontrast für Senioren)
